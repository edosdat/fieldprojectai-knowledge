# fieldprojectai-knowledge

Base de connaissances dédiée à la configuration de projets **Bulldozair** selon la **Méthode Éloi**, utilisée comme référence pour l’agent **FieldProjectAI**.

## Site publié

Documentation en ligne :  
https://edosdat.github.io/fieldprojectai-knowledge/

## Objectif

Ce projet a pour but de centraliser un corpus structuré de règles, méthodes et bonnes pratiques pour :

- configurer un projet Bulldozair
- définir les bons tags et workflows
- structurer le reporting
- cadrer les cas d’usage terrain
- alimenter un agent IA spécialisé

## Sujets couverts

La base de connaissances couvre notamment :

- philosophie de structuration d’un projet
- règle 1 espace = 1 usage
- rôles utilisateurs et groupes
- structure d’une tâche
- familles de tags
- statuts et workflows
- formulaires
- plans PDF, IFC / BIM, SIG / GeoJSON
- reporting
- cas d’usage Suivi de chantier, OPR, FNC, CRR, QSE
- méthode Éloi
- checklists et anti-patterns

## Public visé

Cette documentation s’adresse à :

- chefs de projet
- équipes avant-vente
- équipes delivery
- administrateurs Bulldozair
- référents métiers
- agents IA ou assistants spécialisés dans la configuration de projets

## Structure

Le site principal repose sur un document HTML statique publié via GitHub Pages, afin de faciliter :

- la lecture humaine
- l’indexation par les moteurs de recherche
- l’exploitation par des assistants IA
